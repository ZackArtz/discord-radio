# discord-radio
![Go](https://github.com/bottleneckco/discord-radio/workflows/Go/badge.svg)
Discord music bot, written in Golang

### Requirements
- youtube-dl
- ffmpeg
- [dca](https://github.com/bwmarrin/dca/tree/master/cmd/dca)

### Development
1. Clone
2. Run `$ go mod download`

### Running
`$ go run main.go`
